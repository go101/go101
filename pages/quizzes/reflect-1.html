What does the following program print? 
package main

import "reflect"

type I interface {
	foo()
	Bar()
}

type T struct {}

func (T) foo() {}
func (T) Bar() {}

func main() {
	var t T
	var i I = t
	var x = reflect.ValueOf(t)
	var y = reflect.ValueOf(&i).Elem()
	println(x.NumMethod(), y.NumMethod())
}

Choices:

Answer: 1 2

Run it on Go play.

Key point:

  • the reflect.Value.NumMethod method counts unexported methods for interface types/values, but only counts exported methods for non-interface types/values.
