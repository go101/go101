What does the following program print? 

package main

type A struct {
	g int
}

func (A) m() int {
	return 1
}

type B int

func (B) g() {}

func (B) f() {}

type C struct{
	A
	B
}

func (C) m() int {
	return 9
}

func main() {
	var c interface{} = C{}
	_, bf := c.(interface{f()})
	_, bg := c.(interface{g()})
	i := c.(interface{m() int})
	println(bf, bg, i.m())
}

Choices:

Answer: true false 9

Run it on Go play.

Key points:
  • Field C.A.g and method C.B.g collide, so they are both not promoted.
  • Method C.B.f gets promoted as C.f.
  • Method C.m overrides C.A.m.
