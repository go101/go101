Go 101 Apps & Libs

  • Golds - an experimental Go local docs server, a Go docs generator, and a Go code reader.
  • GoTV - a tool used to install and use coexisting installations of Go toolchain verisons harmoniously.
  • nstd - a single-package go module which provides some missing types and functions in the standard library.
  • go-tmd - a TapirMD library for Go.
